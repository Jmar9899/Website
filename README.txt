This folder is a ready-to-upload static website that meets the required ePortfolio elements:
- Landing page + CV
- Reflection A (Sustainability) with APA 7 references
- Reflection F (Intercultural/Global contexts) with APA 7 references
- STAR-based cover letter addressing selection criteria

How to use with WordPress (recommended if you want a hosted site):
1) In WordPress, create 5 Pages: Landing, Curriculum Vitae, Reflection A, Reflection F, Cover Letter.
2) Copy/paste the content from each HTML page into the WordPress editor (or use the HTML block).
3) Upload the PDF/DOCX files from /downloads to your Media Library, then replace the download links.
4) Add a menu with the 5 pages in the same order as the top navigation.

How to use with Wix:
1) Create a new Wix site (blank template).
2) Create 5 pages matching the files above.
3) Copy/paste the content. For references, keep the citation list.
4) Upload the CV and cover letter PDFs in Wix Media and link the buttons.

Local preview:
- Open index.html in a browser.
